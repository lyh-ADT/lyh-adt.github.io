# 💪 健身计时器

一个专为健身训练设计的React H5应用，帮助你记录训练组数和组间休息时间。

## 功能特性

- ✅ **组数记录**：快速记录每次训练的组数
- ⏱️ **休息计时**：提供30秒、1分钟、90秒、2分钟的快捷休息计时
- 💾 **数据持久化**：使用localStorage自动保存所有训练记录
- 📱 **移动端优化**：完美适配手机屏幕，支持触摸操作
- 📊 **训练历史**：查看所有历史训练记录
- 🎨 **美观界面**：现代化设计，渐变色背景

## 使用方法

1. **开始训练**
   - 在输入框中输入训练动作名称（如：卧推、深蹲）
   - 点击"完成一组 +1"按钮记录每组训练

2. **组间休息**
   - 完成一组后，点击休息时间按钮开始倒计时
   - 可以随时停止休息

3. **完成训练**
   - 训练结束后点击"完成训练"按钮
   - 记录会自动保存到训练历史

4. **查看历史**
   - 在下方"训练历史"区域查看所有记录
   - 可以删除单条记录或清空所有记录

## 技术栈

- React 18
- Vite
- localStorage API
- CSS3

```bash
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 浏览器支持

支持所有现代浏览器，包括：
- Chrome/Edge (最新版本)
- Safari (iOS 12+)
- Firefox (最新版本)

## 数据存储

所有训练数据存储在浏览器的localStorage中，不会上传到任何服务器。数据仅在本地设备上保存。
