# Web Ruler - 网页直尺工具

一个基于 React 的网页直尺工具，支持在屏幕上测量物理尺寸和像素距离。

## 功能特性

- 📏 **多单位支持**: 厘米 (cm)、英寸 (inch)、像素 (px)
- 📱 **移动端适配**: 触摸拖拽、响应式设计
- 🎯 **屏幕校准**: 使用常见参照物校准 DPI
- 🎨 **黑白极简**: 简洁高对比度界面
- 💾 **本地存储**: 自动保存校准数据
- ➕ **多标尺**: 支持同时显示多个标尺

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 预览
npm run preview
```

## 使用说明

1. 打开应用后会自动显示一个水平标尺
2. 拖拽标尺可以移动位置
3. 点击"校准"使用参照物校准屏幕 DPI
4. 点击"+ 添加标尺"添加更多标尺
5. 使用透明度滑块调整标尺透明度

## 技术栈

- React 18
- Vite
- 原生 CSS
